# Inventory Manager

Simple app to manage products. Add, remove, search, save and load from CSV.

## How to use it
```bash
python app.py
```

Pick an option from the menu and follow the instructions.

## What it does
- Add products (name, price, quantity)
- View all products
- Search for a product
- Update product info
- Delete products
- Save/load products to/from CSV file
- Show some stats

## Files
- `app.py` - Main menu
- `services.py` - Functions for products
- `files.py` - CSV stuff
- `db.csv` - Data file
